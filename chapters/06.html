マネーフォワード
AI Cowork
── 日本発、バックオフィス特化

日本では、マネーフォワードが2026年4月7日に「マネーフォワード AI Cowork」を発表し、2026年7月のサービス開始に向けた事前登録の受付を始めた。
Announce
2026年4月7日（日本時間）
Launch
2026年7月予定
Focus
経理・人事・法務のバックオフィス業務
Engine
Claude Agent SDK + Claude API + MCP

経理、人事、法務などのバックオフィス業務を、自然言語チャットで丸ごと任せられる自律型エージェント群を中核に据える。ユーザーが「今月の経理処理、全部やっておいて」のような曖昧な指示をチャットで投げると、「オーケストレーター」と呼ばれる調整役エージェントが意図を解釈し、請求書発行、支払依頼、社内メールのドラフト作成といった一連のタスクを、会計・税務・財務・人事・法務などに特化した専門エージェントに割り振って並列に進める。

単一AIではなく、複数エージェントの協調を正面に据えた、正真正銘のマルチエージェント構成。

日本企業のガバナンス要件に対応

最大の特徴は、バックオフィスというJ-SOXや内部統制が重くのしかかる領域で、企業ガバナンスを両立させる設計をゼロから組み込んでいることである。

  • ガードレール機能AIが常に社内ルールに沿って動作するよう制約する。
  • Draft & ApproveAIが作成したドラフトを人間が最終承認してから確定させる。
  • AI監査ログ「誰が、いつ、どのAIを、どう操作したか」を完全に追跡できる。

MCP基盤のオープン化という地ならし

M
2026年3月26日 「マネーフォワード クラウド会計」のリモートMCPサーバーをすべてのプランで開放。従来プラチナランク以上の専門家事務所限定だったAPI・MCP機能を全ユーザーに解放した。

このMCPサーバーはClaude Desktop、Claude Code、Claude Cowork、Cursor、Gemini CLIなど主要AIツールに対応しており、AIエージェントが仕訳登録、元帳検索、データ突合、レポート作成を画面操作なしで実行できるようになっている。

✎
SEVENRICH会計事務所の声 「ClaudeとSkillsの組み合わせで一部のクライアントについては、データ入力からレビューまでの会計ワークフロー全体の自動化を試行錯誤の末に達成した」

「マネーフォワード AI Cowork」は、このMCPによる地ならしの上に7月に登場する、いわば「上物」に相当する。「マネーフォワード クラウド」の各サービスへの標準統合設定が最初から用意されており、既存顧客はAIサービスを別途契約する必要もない。
