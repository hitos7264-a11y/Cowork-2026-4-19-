Key Terms

AIコワーク時代を読み解くためのキーワード集。検索で即座に目的の語へ。

AI CoworkAIコワーク
AIが、あなたの代わりに、デスクトップやクラウド業務環境の上で、複数ステップの実作業を最後まで自律的にやりきる仕組み。「成果物そのものを届ける」点でチャットボットと区別される。
Orchestratorオーケストレーター
調整役エージェント。ユーザーの意図を解釈し、会計・税務・財務など特化エージェントへタスクを割り振り、並列実行を管理する。マネーフォワード AI Coworkが採用。
Multi-Agentマルチエージェント
特定の役割を持つ複数のエージェントが並列で協調して働く構成。単一エージェント型の順次処理ボトルネックを解消する、AIコワークの第一の柱。
Human-in-the-LoopHITL
重要な判断や破壊的操作は必ず人間の承認を通す設計思想。自律性と統制のバランスを保つための仕組み。
Agent Skillsエージェント・スキル
特定業務に特化した手順や知識をテキストリソースとしてパッケージ化したもの。「Plugin」「Connector」と並ぶ、拡張性を担う仕組み。
MCPModel Context Protocol
AIエージェントと外部ツール・データを接続するためのプロトコル。マネーフォワード AI CoworkはMCP経由で「マネーフォワード クラウド会計」等と連携する。
Work IQ
Microsoft Copilot Coworkが備える、Outlookメール・Teamsメッセージ・M365ファイル・Excel・会議録画を横断的に参照して業務文脈を把握する仕組み。
Draft & Approve
AIが作成したドラフトを人間が最終承認してから確定させるプロセス。マネーフォワード AI Coworkのガバナンス設計の核。
Guardrailガードレール
AIが社内ルールに沿って動作するよう制約するメカニズム。暴走や逸脱を防ぐ「柵」。
RBACロールベースアクセス制御
ユーザーをグループ化し、各グループにカスタムロールを割り当ててアクセスを制御する仕組み。Claude Cowork 4月GAで導入。
OpenTelemetry
ツール呼び出し、ファイル読み書き、スキル利用などAIの全アクションをイベントとして吐き出し、Splunk・CriblなどSIEMに流し込むための仕組み。
SIEM
Security Information and Event Management。セキュリティ情報イベント管理。Splunk、Criblなどが代表例。
Prompt Injectionプロンプトインジェクション
AIが参照するWebページ・画像・メール・文書に悪意ある指示を混ぜ、AIが正規指示と誤認して実行してしまう攻撃。AIコワークの最大リスク。
BYOKBring Your Own Key
自分のAPIキーを持ち込んで利用する方式。Kuse Coworkなどが採用し、モデル非依存・ローカルファーストを実現する。
DIY / DIWM / DIFM
Do It Yourself（自分でやる）／Do It With Me（一緒にやる）／Do It For Me（代わりにやってくれる）。Coworkは本格的にDIFMの領域に到達した。
Taskタスク
Claude Coworkでのセッション単位。従来の「チャット」ではなく、ゴールを持つ作業単位としてセッションが構成される。
CAMEL-AI
Eigent AIが主導するオープンソースのマルチエージェントフレームワーク。GitHubで35,000+ stars、200+ contributors。
CoCEO
CoWork.comが提供するシニアAI戦略家相当のメタエージェント。他のエージェントにタスクを委譲し、毎日の日次レポートを返す「AI参謀」。
PlayLoop
CoWork.com上で実行されるワークフロー単位の呼称。複数プラットフォームのエージェントを組み合わせて回せる。
Playbookプレイブック
組織独自の業務手順やリスク許容度を記述したドキュメント。法務プラグインなどはプレイブックに沿ってカスタマイズされる。
Connectorコネクター
外部SaaSやツールとエージェントを接続する機能。Zoom公式MCPコネクターなど。Coworker AIは100+コネクターを持つ。
Analytics API
Claude Cowork 4月GAで提供。DAU/WAU/MAU、各ユーザーのセッション数、スキル・コネクター呼び出し状況を外部に取り出せる。
Spend Limits支出上限
グループ単位で管理コンソールから設定できる予算上限。部門別の利用管理を可能にする。
Frontier Program
Microsoftが Copilot Cowork を Research Preview より広範な企業に展開するプログラム。2026年3月末開始。
J-SOX
日本版SOX法（金融商品取引法における内部統制報告制度）。マネーフォワード AI Coworkが対応する日本企業のガバナンス要件の核。
SCIM
System for Cross-domain Identity Management。IdPからのユーザー同期の標準プロトコル。Claude CoworkのRBACで利用可能。
Apple Virtualization Framework
macOS上で仮想マシンを作成・実行するためのApple公式フレームワーク。Claude Coworkの「手足」側のVM隔離に使用されている。
Claude Agent SDK
Anthropic公式のエージェント開発キット。マネーフォワード AI Coworkの技術基盤として採用されている。
