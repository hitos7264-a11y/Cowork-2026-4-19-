# Cowork 2026年4月19日

**AIが同僚になる年 ── スマホ特化型WEBガイド**

2026年4月時点の「AI COWORK」をめぐる徹底ガイドを、スマートフォンに完全特化したUIで再編集した静的WEBサイトです。Anthropic Claude Cowork、Microsoft Copilot Cowork、マネーフォワード AI Cowork をはじめとする主要9製品と、15章＋はじめに＋おわりにの合計17本の読み物を、暗黒×ネオン調のモダンなUIで提供します。

---

## 🎯 プロジェクトの目的

- 元の長文テキスト（約52KB）を、スマホで読みやすく、かつ高品質な体験として再設計する
- 章ごとのカラーテーマ・カラウト・引用・キーバリューリストなど、読み物体験を阻害しない豊かな表現を実装
- 目次、年表、プレイヤー一覧、用語集（検索機能付き）など、回遊性の高い複数ビューを提供
- 固定ヘッダー＋固定ボトムナビ＋safe-area 対応で、iOS/Android 両方で快適に読める

---

## ✅ 実装済み機能

- **スマホ完全特化UI**（最大幅480pxのアプリ風レイアウト、safe-area-inset対応）
- **暗黒×ネオン調デザインシステム**（CSS変数による一元管理、Noto Sans JP + Space Grotesk）
- **アニメーション付きグラデーション背景**（放射グラデ＋グリッドパターン＋ゆっくりドリフト）
- **固定ボトムナビゲーション**（ホーム／年表／製品／用語／読む の5タブ）
- **スクロール連動の要素フェードイン**（IntersectionObserver）
- **数値カウントアップ**アニメーション（ホーム画面のスタッツ）
- **グラデーションタイトル**（連続的に色相が変化）
- **用語集の全文検索**（インクリメンタルフィルタ）
- **章ページの前後ナビゲーション**（全17本間をシームレスに往復可能）
- **カラーテーマ別バッジ**（violet/cyan/pink/mint/amber）
- 画像を一切使用しない純CSS＋SVGアイコンのみの構成
- **進捗バーは導入していません**（要件通り）

## 🎨 デザインシステム（`css/style.css`）

- **カラー**
  - 背景: `#07080c`（deep space）
  - アクセント: violet `#7c5cff`, cyan `#22d3ee`, pink `#f472b6`, mint `#34d399`, amber `#fbbf24`
  - グラデーション `--grad-primary` `--grad-text` を複数の要素で共有
- **タイポグラフィ**
  - 日本語: Noto Sans JP 400/500/600/700/800
  - 英字・数字: Space Grotesk 400/500/600/700
- **レイアウト**
  - `.app { max-width: 480px }` でスマホ幅固定
  - PC表示時はアプリ風にセンタリング＋外側を完全暗転
- **コンポーネント**
  - `.chapter-card` `.pull-quote` `.callout` `.feature-list` `.kv-list` `.player-card` `.term` `.tl-item` などを再利用可能に設計
